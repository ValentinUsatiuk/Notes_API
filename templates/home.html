Main URL Paths

User Registration

  • URL: /register
  • Method: POST
  • Parameters: username (username), password (password)
  • Description: Register a new user in the system.
  • Example Request:
    • {
  "username": "example_user",
  "password": "password123"
}
  • Example Response:
    • {
  "message": "User registered successfully"
}

User Login

  • URL: /login
  • Method: POST
  • Parameters: username (username), password (password)
  • Description: Authenticate a user in the system.
  • Example Request:
    • {
  "username": "example_user",
  "password": "password123"
}
  • Example Response:
    • {
  "message": "Login successfully"
}

List All Notes

  • URL: /notes
  • Method: GET
  • Parameters: None
  • Description: Get a list of all notes in the system.
  • Example Response:
    • {
  "notes": [
    {
      "id": 1,
      "title": "Note Title",
      "content": "Note Content",
      "created_on": "2023-09-10 14:30:00",
      "user_id": 1
    },
    {
      "id": 2,
      "title": "Another Note",
      "content": "Text of another note",
      "created_on": "2023-09-11 10:15:00",
      "user_id": 1
    }
  ]
}
